# Redux Thunk E-Commerce

A functional E-Commerce application demonstrating asynchronous state management using **Redux Thunk**. This project focuses on handling complex shopping cart logic and side effects (API calls) within the classic Redux architecture.

![ecommerce](https://github.com/user-attachments/assets/40471e7c-fe65-46e9-b6b8-cf52e887263a)


## Key Features

* **Redux Middleware:** Asynchronous data fetching using **Redux Thunk**.
* **Cart Management:** Add to cart, remove items, and calculate totals globally.
* **API Integration:** Fetching product data via **Axios**.
* **Routing:** Dynamic page navigation using **React Router v6**.
* **Responsive UI:** Clean interface styled with **Bootstrap 5**.

## Tech Stack

| Category | Technologies |
| :--- | :--- |
| **Framework** | React 18 (Vite) |
| **State Management** | Redux, Redux Thunk |
| **Styling** | Bootstrap 5 |
| **Routing** | React Router DOM v6 |
| **API Client** | Axios |
| **Mock Backend** | JSON Server |
